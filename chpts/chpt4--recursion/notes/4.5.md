# 4.5 Designing Recursive Algorithms
- In general, an algorithm that uses recursion typically has the following form:
    - **Test for base cases**:
        - Begin with testing for a set of base cases. These need to be defined so that every possible chain of recursive calls has an eventual end when reaching base cases
    - **Recur**:
        - If not a base case, performs recursive calls -> this may involve a test that decides which of several possible recursive calls to make. Each possible recursive call progresses towards a base case
- **Parameterizing a Recursion**:
    - To design a recursive algorithm for a given problem
        - Think of the different ways sub-problems may be defined so that they have the same structure as the original problem
