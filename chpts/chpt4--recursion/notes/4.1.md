## 4.1.1 The Factorial Function
- $n!$ -> 1 if $n = 0$; $n * (n - 1) * (n - 2) * ... 1$ if $n \geq 1$
    - The recursive definition:
        - $n! = n * (n - 1)!$ if $n \geq 1$
- **A Recursive Implementation of the Factorial Function**:
    - Recursive design of a factorial function
    ```Python
    def factorial(n):
        # base case
        if n == 0:
            return 1
        else:
            # recursive case
            return n * factorial(n - 1)
    ```
    - In python each time a fucntion is called, a structure known as an `activation record` or `frame` is created to store information about the progress of that invocation of the function
        - This `activation record` includes a namespace for storing the function call's params and local vars and information about which cmd in the body of the function is curr executing

## 4.1.2 Drawing an English Ruler
- Recursive breakdown:
    - Each big tick has smaller
```Python
# base line
def draw_line(tick_length, tick_label = ""):
    """Draw one line with given tick length (followed by optional label)"""
    line = "-" * tick_length

    if tick_label:
        line += " " + tick_label
    print(line)

# each interval
def draw_interval(center_length):
    """draw tick interval based upon a central tick length"""
    if center_length > 0:
        # -
        # --
        # -
        draw_interval(center_length - 1)
        draw_line(center_length)
        draw_interval(center_length - 1)

# entire ruler
def draw_ruler(num_inches, major_length):
    """draw ruler with given number of inches and major tick length"""
    draw_line(major_length, "0")

    for j in range(1, 1 + num_inches):
        draw_interval(major_length - 1)
        draw_line(major_length, str(j))
```

## 4.1.3 Binary Search
```Python
def binary_search(data, target, low, high):
    """
    Return True if target is found in indicated portion of a Python list

    Only considers the portion from low to high inclusive
    """
    if low > high:
        return False
    else:
        mid = (low + high) // 2

        if target == data[mid]:
            return True
        elif target > data[mid]:
            return binary_search(data, target, mid + 1, high)
        elif target < data[mid]:
            return binary_search(data, target, low, mid - 1)
```

- **Python's os Module**:
    - Tools for interacting with the operating system during the execution of a program
    - Four most important functions:
        - `os.path.getsize(path)`:
            - return the immediate disk usage (measured in bytes) for the file or directory that is identified by the string path--i.e. /user/rt/courses
        - `os.path.isdir(path)`:
            - return True if entry designated by string path is a directory else False
        - `os.listdir(path)`:
            - return a list of strings that are the names of all entries within a directory designated by string path
            - returns a list--`['dirone', 'dirtwo']`
        - `os.path.join(path, filename)`
            - compose the path string and filename string using an appropriate operating system separator between the two--i.e. the `/` character for a Unix/Linux system
            - returns the string that represents the full path to the file
    
    - **Python Implementation**:
    ```Python
    import os

    def disk_usage(path):
        """Return the number of bytes used by a file/folder and any descendents"""
        
        # direct folder size
        total = os.path.getsize(path)

        # if path is a folder
        if os.path.isdir(path):
            # iterate through folder
            for filename in os.listdir(path):
                # create path for each item in folder
                childpath = os.path.join(path, filename)
                # recursively call disk_usage on files/folders in og path and add it to total size
                total += disk_usage(childpath)
        
        # print first 7 digits of the total size and the path provided of the folder
        print('{0:<7}'.format(total), path)
        return total
    ```
