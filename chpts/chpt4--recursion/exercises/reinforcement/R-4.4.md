## Draw the recursion trace for the execution of function `reverse(S, 0, 5)` on $S = [4, 3, 6, 2, 6]$
```Python
def reverse(S, start, stop):
    if start < stop - 1:
        S[start], S[stop - 1] = S[stop - 1], S[start]
        reverse(S, start + 1, stop - 1)
```
