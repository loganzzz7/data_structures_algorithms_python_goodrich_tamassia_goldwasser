## Draw the recursion trade for the computation of power(2, 18) using the repeated squaring algorithm
```Python
def power(x, n):
    if n == 0:
        return 1
    else:
        partial = power(x, n // 2)
        res = partial * partial
        if n % 2 != 0:
            res = res * x
    return res
```
