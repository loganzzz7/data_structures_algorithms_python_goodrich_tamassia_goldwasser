## Draw the recursion trace for the execution of function `PuzzleSolve(3, S, U)` where $S$ is empty and $U = \{a, b, c, d\}$
```Python
Algorithm PuzzleSolve(k, S, U):
    """
    Input: an integer k, sequence S, set U
    Output: an enumeration of all k-length extensions to S using elts in U without repetitions
    """
    for each e in U do:
        add e to the end of S # e is now being used so remove from U
        U.remove(e)
        if k == 1 then:
            test whether S is a configuration that solves the puzzle
            if S solves the puzzle then:
                return "Solution found: " S
        else:
            # recursive call with trying next int
            PuzzleSolve(k - 1, S, U)
        # e didnt work so remove e from answer S and add it back to unused set U
        S.remove(e)
        U.add(e)
```
