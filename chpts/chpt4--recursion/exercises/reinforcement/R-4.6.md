## Describe a recursive function for computing the $n^{th}$ **Harmonic number**, $H_n = \sum^n_{i = 1}\frac{1}{i}$
- Base Case: n == 1 -> return $\frac{1}{i = 1} = 1$
- Recurse -> `return` $\frac{1}{n} +$ `find_harmonic(n - 1)`
```Python
def find_harmonic(n):
    """Return the n-th harmonic number"""
    if n == 1:
        return 1
    else:
        return 1/n + find_harmonic(n - 1)
```
