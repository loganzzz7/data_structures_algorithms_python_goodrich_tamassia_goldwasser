## Describe a recursive function for converting a string of digits into the integer it represents. For example, '13531' represents the integer `13,531`
- WANT TO CONVERT STRING TO NUMBER, GETTING EACH DIGIT VIA BASE 10
    - Idea: get every digit besides last via * 10 and append last digit
- Base Case: `if len(S) == 1: return int(S)`
- Recurse: `return s_to_i(S[:len(S) - 1]) * 10 + int(S[len(S) - 1])`

Code:
```Python
def s_to_i(S):
    if len(S) == 1:
        return int(S)
    else:
        return s_to_i(S[:len(S) - 1]) * 10 + int(S[len(S) -  1])
```
```
Recursive Stack:
s_to_i("13531")
=> s_to_i("1353") * 10 + 1
=> (s_to_i("135") * 10 + 3) * 10 + 1
=> ((s_to_i("13") * 10 + 5) * 10 + 3) * 10 + 1
=> (((s_to_i("1") * 10 + 3) * 10 + 5) * 10 + 3) * 10 + 1
= 13531
```
