# 1.6 Simple Input and Output
## 1.6.1 Console Input and Output
**OUTPUT**
- print() -> output to the console
    - has a default vars that can be user-defined (below is the default)
        - i.e. `print(1, 2, 3, sep=' ', end='\n')`

**INPUT**
- input() -> get user input from the console -> default gets it in str -> can convert to other types after
    - i.e. `age = int(input('Type your age: '))`
    - another
```python
reply = input('enter x and y separated by spaces: ')
res = reply.split(" ")
x = res[0]
y = res[1]
```

**FILE I/O**
- accessed with the built-in function `open()`
    - i.e. `fp = open('sample.txt', mode='r')`
        - default mode == 'r'
        - other modes: 'w' = write--this overwrites the file; 'a' = append--this adds at the end of the file;
        - binary: 'rb' = read binary; 'wb' = write binary; 'ab' = append binary;
```
Calling Syntax -> Desc:
fp.read() -> return the remaining contents of the file as str

fp.read(k) -> return the next k bytes of a readable file as str

fp.readline() -> return the remainder of the curr line as str

fp.readlines() -> return all remaining lines as a list of strings

fp.seek(k) -> change curr pos to be at the k-th byte of the file

fp.tell() -> return the curr pos, measured as byte-offset from the start

fp.write(str) -> write string at curr pos of file

fp.writelines(seq) -> write each strs of given seq at curr pos of file--DOES NOT insert newlines if not provided in the strs of the given seq

print(..., file=fp) -> redirect output of print func to file!

fp.close() -> closes file and saves written contents
```
