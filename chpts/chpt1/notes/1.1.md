# 1.1 Python Overview
- Python 3.1+: high level language. Overview of python programming languauge.
## 1.1.1 Python Interpreter
- Python is an interpreted language -> cmds are executed though the Python Intrepreter.
    - The python interpreter receives a command, evaluates that command and reports the rsult of the command.
    - Programmer defines a series of commands in advance and saves those commands in a plain txt file known as the source file or script.
    - Source code is stored in a file named with the .py suffix
    - Integrated development environments (IDEs) provide better software development platforms for Python
