# 1.5 Functions
- Defined via `def`
    - Each time a function is called -> Python creates a dedicated **activation record** that stores info relevant to the current call -> includes the curr namespace and all the identifiers will have local scope
- Default Parameter Values: python supports polymorphic functions -> supports more than one calling signature
    - functions can declare default vars with given vals and then can be called without providing the default given vars
        - i.e. `foo(a, b = 19, c = 20)` can be called w either `foo(1, 2, 3) or foo(1)` where b and c would be the default vals
        - can also specify which var given a func with default -> `foo(a = 1, b = 2)` then c remains default
- Common Built-In Functions: [linked](https://docs.python.org/3/library/functions.html)
