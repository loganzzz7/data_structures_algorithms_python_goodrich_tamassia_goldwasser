# 1.11 Modules and the Import Statement
- Python's **import** stmt loads defs from a module into the curr namespace
    - i.e. `from math import pi, sqrt`
        - This cmd adds both pi and sqrt as defined in the math module into the curr namespace! then can directly use pi and sqrt
    - i.e. `import math` -> can also import the entire module
- Creating a new module!!
    - import func from another file `name.py` via `from name import func`.
Existing Modules:
```
module name -> desc
array -> compact array storage for primitive types
collections -> additional data structures and abstract base classes involving collection of objects
copy -> generatl functions for making copies of objs
heapq -> heap-based prio queue
math -> common math constants and funcs
os -> interactions with the operating system
random -> random num generations
re -> regex
sys -> additional level of interaction with the python interpreter
time -> support for measuring time, or delaying a program
```

Random Class breakdown:
```
syntax -> desc
seed(hashable) -> init pseudo-random number generators based upon the hash value of the param
random() -> returns a pseudo-random floating-point value in interval [0.0, 1.0)
randint(a, b) -> returns a random int in closed interval [a, b]
randrange(start, stop, step) -> return a pseudo-random int in the standard Python range indicated by the params
choice(seq) -> rets an elt of the given seq chosen pseudo-randomly
shuffle(seq) -> reorders the elts fo the given seq randomly
```
