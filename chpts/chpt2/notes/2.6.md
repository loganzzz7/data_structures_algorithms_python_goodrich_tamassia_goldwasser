# 2.6 Shallow and Deep Copying
- In chpt 1, learned that `foo = bar` makes the name `foo` an **alias** of the object identified as `bar`
    - This section detailed the task of making a **copy** of an object, rather than an alias
        - To create a **copy** -> need to create a new instance of the original
            - i.e. if we want to create a copy of a list rather than an alias to the same list, need to do 
                - `new = List(original)`
                - `new = original` makes `new` and alias to the `original` list object
        - however `new = List(original)` is considered a **shallow copy** since the new list represents a sequence of references to the same elements as in the first
            - a **deep copy would be completely different references sharing the same values
                - to create a **deep copy** -> use the Python built-in `copy` module 
                - i.e. `new = copy.deepcopy(original)`
