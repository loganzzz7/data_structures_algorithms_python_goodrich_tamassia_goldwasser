# 3.2 The Seven Functions Used in This Book
- **The Constant Function**:
    - `f(n) = c`
        - for some fixed constant `c`, such as `c = 5, c = 27, c = 2^10`
        - for any argument `n`, the constant function `f(n)` assigns the value `c` -> it does not matter the what the value of `n` is, `f(n) = c` since `n` plays no role on the RHS
- **The Logarithm Function**:
    - `f(n) = log_b(n)` for some constant b > 1 -> log def: `x = log_bn iff b^x = n` -> by def `log_b1 = 0; b^0 = 1`
        - since binary is base 2 -> in algorithms with common operation (divide & conqueror), the base is 2 -> `logn = log_2n`
    - **Logarithm Rules**:
        1. $log_b(ac) = log_ba + log_bc$
        2. $log_b(a/c) = log_ba - log_bc$
        3. $log_b(a^c) = clog_ba$
        4. $log_ba = \frac{log_da}{log_db}$
        5. $b^{log_da} = a^{log_db}$
- **The Linear Function**:
    - $f(n) = n$
        - given an input value $n$, the linear function $f$ assigns the value $n$ itself
- **The N-Log-N Function**:
    - $f(n) = nlogn$
        - The function that assigns to an input $n$ the value of $n$ times the logarithm base-two of $n$
        - This function grows a little more rapidly than the linear function but less than the quadratic function
- **The Quadratic Function**:
    - $f(n) = n^2$
        - Given an input value $n$, the function $f$ assigns the product of $n$ with itself -> `n squared` so quadratic

- **Nested Loops and the Quadratic Function**:
    - The quadratic function arises in the context of nested loops

- **Polynomial time**:
    - all forms of $n^d$ where $d$ is the degree of the polynomial

- **Summations**:
    - $\sum^b_{i = a} f(i) = f(a) + f(a + 1) + f(a + 2) + ... + f(b)$
        - summations arise in data structure and algorithms because the running times of loops naturally give rise to summations
    - rewritten as $\sum^n_{i = 1} = \frac{n(n+1)}{2}$
    - Also written as: $f(n) = \sum^d_{i = 0}a_in^i$

- **Exponential function**: 
    - $f(n) = b^n$
        - exponential time

- **Geometric Sum**:
    - Suppose a loop for which eac iteration takes a multiplicative factor longer than the previous one
        - $\sum^n_{i = 0}a^i = 1 + a + a^2 + ... + a^n$
        - This is eq to $\frac{a^{n + 1} - 1}{a - 1}$
- **Growth Rate Comparison--Best to Worst**:
    1. constant = 1
    2. logarithm = logn
    3. linear = n
    4. n-log-n = nlogn
    5. quadratic = n^2
    6. cubic = n^3
    7. exponential = a^n
